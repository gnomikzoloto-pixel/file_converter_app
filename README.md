# File Converter App

📌 О проекте

Простое и красивое веб-приложение, которое позволяет конвертировать файлы между тремя популярными форматами:

- **CSV** (табличные данные)
- **JSON** (обмен данными)
- **XML** (структурированные данные)

**Ключевые особенности:**
- 🎨 Красивый современный интерфейс с поддержкой drag-and-drop
- 📋 Выпадающий список форматов (не нужно вводить вручную)
- 👁️ Предпросмотр результата конвертации
- 📥 Скачивание результата одним кликом
- 🗑️ Кнопка очистки выбранного файла
- ⚡ Асинхронная обработка файлов
- 📚 Автоматическая Swagger документация API

---

## 🚀 Быстрый старт

### Требования
- Python 3.10 или выше
- pip (менеджер пакетов)

### Установка и запуск

```bash
# 1. Клонировать репозиторий
git clone https://github.com/gnomikzoloto-pixel/file_converter_app.git
cd file_converter_app

# 2. Создать виртуальное окружение
python -m venv venv

# 3. Активировать окружение
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# 4. Установить зависимости
pip install -r requirements.txt

# 5. Запустить сервер
python run.py
